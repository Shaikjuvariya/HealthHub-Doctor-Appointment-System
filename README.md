# 🏥 HealthHub - Doctor Appointment System

## 📌 Project Overview

HealthHub is a Full Stack Doctor Appointment System developed using Spring Boot, React, and MySQL. It allows patients to register, log in, view doctors, and book appointments. The system also provides an Admin Dashboard to manage doctors and monitor appointments.

---

## 🚀 Features

### 👤 Patient
- User Registration
- User Login
- View Doctors
- Book Appointment
- Success & Thank You Pages

### 👨‍💼 Admin
- Admin Login
- Dashboard
- View All Appointments
- Add Doctors
- View Statistics
- Manage Doctors

## ⚙️ Installation

### Backend

1. Open the backend project.
2. Configure MySQL in `application.properties`.
3. Run the Spring Boot application.

### Frontend

```bash
npm install
npm start
```
